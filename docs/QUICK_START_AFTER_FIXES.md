## 🐛 Troubleshooting

### If you see import errors:
```bash
cd client
npm install
```

### If themes don't change:
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh the page (Ctrl+F5)

### If splash screen doesn't show:
- Check browser console (F12)
- Ensure framer-motion is installed
- Refresh the page
